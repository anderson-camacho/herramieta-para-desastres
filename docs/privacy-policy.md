# Politica de privacidad

SignalScope procesa la informacion localmente en el dispositivo para mostrar observaciones de senal celular, Wi-Fi, Bluetooth y compatibilidad USB Host.

## Informacion procesada

- Estado de permisos
- Estado de hardware y servicios del sistema
- Valores tecnicos aproximados de senal como RSSI o dBm cuando Android los expone
- Historial local opcional de puntos de senal

## Informacion no recopilada

- IMEI
- IMSI
- Numero de telefono
- Mensajes
- Contactos
- Archivos personales
- Claves Wi-Fi

## Almacenamiento

El MVP no envia datos a un backend. El historial se guarda localmente solo si el usuario lo mantiene habilitado.

## Exportacion y eliminacion

Los logs locales y el historial podran limpiarse desde la aplicacion conforme evolucione el modulo de diagnostico. En este MVP el historial puede desactivarse y deja de acumularse.

## Contacto
